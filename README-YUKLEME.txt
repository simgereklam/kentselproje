Kentsel Dönüşüm Firma Başlık Tek Tip Paket

Bu pakette sadece üst alan tek tip yapıldı:
- En üstteki İstanbul geneli / WhatsApp yazı satırı kaldırıldı.
- Header logosu yerine MASFEY İNŞAAT ana başlığı eklendi.
- Alt satırda Kardelen İnşaat, Örnek Yapı İnşaat, Lavinay İnşaat ve Korkut Mobilya Grup İştirakleri yazıyor.
- Menü, mobil yapı, sayfa içerikleri, görsel büyütme/lightbox ve diğer alanlar korunmuştur.

GitHub ana klasörüne zip içindeki tüm dosyaları yükleyip eskilerin üzerine yazdırın.
Özellikle tüm .html dosyaları ve style.css değişmiştir.


SEO + DİKKAT ÇEKİCİ DÜZENLEME NOTU
----------------------------------
Bu paket mevcut tasarımı değiştirmez.
Sadece şu işlemler eklenmiştir:
- Title ve meta description güçlendirildi.
- Canonical URL'ler korundu ve güncellendi.
- Open Graph ve Twitter kart etiketleri eklendi.
- LocalBusiness / WebSite / WebPage schema güçlendirildi.
- S.S.S sayfasına FAQ schema eklendi.
- Sitemap lastmod tarihi 2026-06-30 yapıldı.
- Görseller için loading="lazy" ve decoding="async" eklendi.
- Ana sayfaya küçük dikkat çekici vurgu satırı eklendi.
- Butonlara çok hafif dikkat çekici animasyon eklendi.
- Menü, sayfa yapısı, görseller ve eski çalışan düzen korunmuştur.

Yükleme:
ZIP içindeki dosyaları GitHub ana dizinine yükleyin ve aynı isimli dosyaların üzerine yazdırın.
